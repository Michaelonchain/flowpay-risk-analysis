# FlowPay (FPAY) — Risk Analysis

## Overview

This document identifies potential risks in the FlowPay token model
and how they can impact long-term sustainability.

---

## Key Risks

### 1. High Emission Pressure
If emissions remain too high for too long:
→ constant sell pressure
→ price decline

Mitigation:
- declining emission schedule
- reward locking mechanisms

---

### 2. Low User Participation
If incentives are not attractive:
→ low adoption
→ weak network activity

Mitigation:
- balanced APR
- reward active users

---

### 3. Short-Term Farming Behavior
Users may farm rewards and sell immediately.

Mitigation:
- lock-based rewards
- reduced rewards for no lock

---

### 4. Centralization Risk
Too many tokens controlled by few holders.

Mitigation:
- fair distribution
- gradual vesting

---

## Conclusion

Strong token design requires identifying risks early
and structuring incentives to minimize them.
